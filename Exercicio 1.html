Tags H1 a H6

Estas tags são utilizadas para representar títulos de seções no código HTML. Por padrão, a tag h1 representa o maior texto, que vai diminuindo até se chegar ao h6. Essas tags possuem correspondentes para fechamento.

Tag p

A tag p é utilizada para descrever parágrafos. Esta tag fornece quebras de linha automaticamente. Esta tag possui correspondente para fechamento.

Tag abbr

abbr

A tag abbr é utilizada para definições de abreviaturas na página. Com a ajuda do atributo title, é possível definir o significado da abreviatura. É possível visualizar tal significado passando o mouse sobre o texto

Tag cite

Cite - para colocar uma citação. Por padrão, o texto fica com o formato itálico. Segue um exemplo:
"Vencer, vencer, vencer este é o nosso ideal..."

Tag hr

A tag hr é utilizada para desenhar uma linha horizontal na página. Esta tag possui fechamento próprio.
